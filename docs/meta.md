# Expanding the Knowledge Base

To add more categories or tutorials, follow these steps:

## Adding a Category
1. Create a new folder under `docs/`.
2. Add a `_category_.json` file in that folder to define its label and position.
3. Add markdown files inside the folder.

## Adding a Tutorial
1. Create a new `.md` or `.mdx` file in any of the category folders.
2. Docusaurus will automatically pick it up and add it to the sidebar.

## Previewing Changes
Run `bun run start` to see your changes live.
